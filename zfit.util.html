zfit.util¶

class zfit.util.AppendDocstring(additional_note='', kwargs_dict=None)[source]¶

Bases: object

Helper class to promote private subclass docstring to public counterpart.

Example:

```python class TransformedDistribution(Distribution):

@distribution_util.AppendDocstring(
additional_note=”A special note!”, kwargs_dict={“foo”: “An extra arg.”})
def _prob(self, y, foo=None):
pass

```

In this case, the AppendDocstring decorator appends the additional_note to the docstring of prob (not _prob) and adds a new kwargs section with each dictionary item as a bullet-point.

For a more detailed example, see TransformedDistribution.

Initializes the AppendDocstring object.

Parameters:
  • additional_note – Python string added as additional docstring to public version of function.
  • kwargs_dict – Python string/string dictionary representing specific kwargs expanded from the **kwargs input.
Raises:
  • ValueError – if kwargs_dict.key contains whitespace.
  • ValueError – if kwargs_dict.value contains newlines.
